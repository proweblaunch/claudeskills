# Typography Director

## Mission

Direct typography as a central brand system.

Typography is not just font choice.
It is hierarchy, tone, rhythm, spacing, readability, and brand character.

This skill should help Claude make type decisions that elevate a website from acceptable to premium.

## What this skill is for

Use this skill when the user wants help with:

- font pairing
- type scale design
- hierarchy planning
- editorial polish
- readability improvements
- premium brand tone through typography
- typographic rhythm for a website
- text-heavy layouts that still feel elegant

## When to use the skill

Use this skill when typography affects the quality of the design, especially for:

- premium landing pages
- SaaS websites
- brand websites
- editorial sections
- hero messaging
- narrative content
- trust-heavy messaging
- any page where type is a major visual component

## When not to use the skill

Do not use this skill when the request is only a simple copy edit with no design implications.

## Core philosophy

Type should do more than convey information.

It should shape tone, confidence, and visual identity.

Premium typography usually feels:

- deliberate
- balanced
- disciplined
- calm
- modern
- confident
- highly legible
- emotionally aligned with the brand

## Typography responsibilities

This skill should guide:

- font selection
- weight distribution
- heading rhythm
- body text comfort
- line length
- spacing between elements
- emphasis rules
- editorial structure
- text hierarchy across sections

## Hierarchy logic

A strong type system should quickly tell the user:

- what matters most
- what comes next
- what is supporting information
- what is meant to persuade or reassure

Hierarchy should feel natural and effortless.

## Premium type characteristics

Strong premium typography often has:

- clear distinction between levels
- generous but controlled spacing
- purposeful line height
- restrained font variety
- consistent alignment behavior
- thoughtful contrast between headline and body styles

## Voice and tone

Typography should support the brand voice.

For example:

- enterprise software should feel precise and credible
- luxury brands should feel refined and composed
- futuristic brands should feel sharp and modern
- editorial brands should feel sophisticated and readable

## Quality standard

If the typography feels generic, the whole website often feels generic.

If the typography feels premium, the rest of the page gets easier to trust.

When in doubt, improve the type system before adding more visual effects.
