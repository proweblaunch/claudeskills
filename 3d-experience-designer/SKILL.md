# 3D Experience Designer

## Mission

Design immersive 3D web experiences for premium brands and futuristic interfaces.

The goal is not to add 3D as a visual gimmick.

The goal is to use 3D to make the website feel like a believable environment with depth, motion, presence, and spatial logic.

A strong 3D experience should make the user feel like they are moving through a constructed world that has layers, atmosphere, and intentional focus.

## What this skill is for

Use this skill when the user wants any of the following:

- a 3D website direction
- immersive spatial design
- floating objects or layered depth
- camera-based website storytelling
- 3D motion concepts for a landing page or homepage
- premium environment design for a futuristic brand
- a scene-based web experience rather than a flat layout
- visual direction that uses depth, lighting, and motion as storytelling tools

## When to use the skill

Use this skill when the request includes:

- 3D website concept
- WebGL, Three.js, or R3F style thinking
- spatial composition direction
- futuristic brand world-building
- floating interface elements
- cinematic object movement
- layered scenes that feel interactive
- premium digital environment design

## When not to use the skill

Do not use this skill when the request is only for:

- a simple card layout
- a static page redesign with no spatial ambition
- a pure copy edit
- a plain implementation fix unrelated to 3D or depth
- a generic UI task where the visual direction is intentionally flat

If the user is not asking for spatial or cinematic direction, this skill is unnecessary.

## Core design philosophy

3D should be used to deepen story, hierarchy, and immersion.

Not every website needs full 3D.

But when 3D is appropriate, it should feel like part of the brand identity rather than an added effect.

A premium 3D experience usually creates one or more of these outcomes:

- stronger sense of scale
- better emotional impact
- clearer hierarchy through depth
- more memorable transitions
- richer storytelling
- a stronger premium perception
- a more futuristic and technically advanced impression

## First principle: scene thinking

Treat each section as a scene inside a world.

Think in terms of:

- foreground
- midground
- background
- camera angle
- object scale
- depth separation
- motion layers
- light direction
- spatial tension
- atmospheric perspective
- focal emphasis
- transition path

If a design does not work as a scene, it will not work as a premium 3D experience.

## Spatial composition

Use depth to create hierarchy and emotion.

Preferred forms include:

- floating panels
- architectural frames
- glass structures
- chrome or metallic elements
- abstract geometric sculptures
- digital rings
- neural or network motifs
- particles
- holographic cards
- data nodes
- light beams
- wireframe systems
- symbolic product forms
- layered product modules
- translucent data surfaces
- orbiting interface fragments

Each form should feel like it belongs to the same environment.

Do not combine random 3D elements just because they look interesting individually.

They must feel connected by a single visual language.

## Object behavior

Objects should feel as if they belong to the same environment.

Give them:

- position
- mass
- rhythm
- rotation
- drift
- response to scroll or cursor
- visual purpose
- material identity
- relationship to the camera

Objects should support the story, not sit there as decoration.

Each object should have a reason to exist:

- to create depth
- to guide attention
- to establish mood
- to support a transition
- to symbolize a brand idea
- to reinforce the premium tone
- to help the user understand the system

If an object does not contribute, remove it.

## Camera thinking

Always ask:

- Where is the camera?
- What is the camera looking at?
- How does the camera move?
- What does the camera reveal?
- What should remain hidden until later?
- What emotional effect does the camera movement create?

Camera motion is one of the strongest tools for making a site feel cinematic.

Camera movement should not be random.

It should behave like a deliberate reveal system.

Use camera ideas such as:

- push-in
- pull-back
- orbit
- dolly
- crane-like rise
- lateral shift
- subtle drift
- focus changes
- small rotations that create spatial tension
- reveal-by-movement rather than reveal-by-fade

## Camera and storytelling

Camera movement should reinforce the story arc.

For example:

- a push-in can communicate importance or arrival
- a pull-back can reveal the wider world
- an orbit can communicate sophistication or inspection
- a lateral shift can move from one idea to another
- a focus change can tell the user what matters now

The camera should feel like it is thinking with the brand.

## Lighting direction

Lighting should reinforce premium quality and depth.

Recommend lighting that feels:

- controlled
- soft where needed
- dramatic where needed
- high-contrast when appropriate
- atmospheric
- polished
- intentional

Lighting should help separate layers, define surfaces, and increase realism.

Use light to shape emotion:

- soft light can feel refined and calm
- hard contrast can feel dramatic and technical
- rim lighting can separate objects from the background
- glow can feel futuristic
- shadow can create weight and seriousness

Lighting should never feel accidental.

## Material direction

When appropriate, favor materials such as:

- glass
- frosted glass
- chrome
- brushed metal
- translucent surfaces
- reflective surfaces
- soft luminous gradients
- matte dark surfaces for contrast
- emissive lines or accents
- polished surfaces that catch light cleanly

Materials should feel intentional and aligned with the brand tone.

Choose materials that support the emotional message:

- glass for clarity and premium transparency
- chrome for precision and technology
- matte dark surfaces for seriousness and contrast
- translucent materials for futurism and lightness

## Visual effects awareness

When the experience needs it, recommend effects such as:

- bloom
- glow
- depth of field
- refraction
- reflection
- volumetric light
- atmospheric haze
- soft shadows
- particle motion
- motion blur
- subtle chromatic styling when appropriate

Use effects in service of depth and storytelling.

Do not overload the scene.

If an effect helps the scene feel richer but hurts clarity, reduce it.

## Scene design logic

Each section should answer:

- what is the central object or idea?
- what is the viewer meant to notice first?
- how does the viewer move through the scene?
- what is the transition into the next scene?
- how does the scene evolve while the user scrolls?
- what spatial relationship is most important here?
- what emotional tone should the scene create?

Good 3D web design always balances beauty, clarity, and control.

A scene can be dramatic, but it must still be understandable.

## Spatial hierarchy

Use depth to support hierarchy.

The most important objects should generally be the clearest, most stable, or most visually dominant.

Supporting elements should sit farther back, move more subtly, or feel more atmospheric.

Good spatial hierarchy helps the user understand what to read, what to feel, and what to remember.

## Scene composition patterns

Useful composition patterns include:

- central hero object with surrounding contextual elements
- layered floating panels with depth-based emphasis
- corridor-like scene that creates forward motion
- orbiting content ring around a central concept
- split depth composition with foreground and background storytelling
- object cluster that reveals information progressively
- exhibition-style scene with controlled spotlighting
- product-as-monument composition

Choose the pattern that best serves the story.

## Spatial transitions

Transitions should feel like movement through the same world.

Use transitions that are spatial rather than abrupt.

Examples:

- camera glide into the next scene
- objects drifting out while new objects come into focus
- lighting shift that signals a new chapter
- depth shift that opens the next section
- scene overlap that keeps continuity

Do not reset the viewer’s sense of place unless there is a strong storytelling reason.

## Interaction design in 3D space

If the concept includes user interaction, keep it subtle and premium.

Possible interactions include:

- slight cursor parallax
- object response to scroll direction
- light response to mouse movement
- hover-based depth changes
- interactive object rotation
- focus-based emphasis shifts

Interactions should feel like the world responds intelligently.

They should not feel like a novelty demo.

## Premium 3D standards

A good premium 3D experience is:

- visually coherent
- spatially believable
- narratively useful
- polished in lighting and materials
- mindful of motion and rhythm
- supportive of content clarity
- technically realistic enough to build well
- memorable without being chaotic

Avoid the following:

- random 3D clutter
- too many competing focal points
- objects that appear disconnected from the story
- materials that feel cheap or noisy
- overuse of effects that reduce legibility
- scenes that are beautiful but confusing

## Common failure modes

Watch out for these issues:

- depth used only for spectacle
- motion with no narrative reason
- too many objects in the scene
- weak camera logic
- lighting that does not support depth
- materials that clash with the brand tone
- transitions that feel disconnected
- 3D complexity that overwhelms the content
- designs that are visually rich but emotionally flat

## How to refine a 3D concept

When the concept feels weak, check:

1. Is there a clear central idea or object?
2. Is the camera helping tell the story?
3. Is depth being used to clarify hierarchy?
4. Do the materials match the brand tone?
5. Are the transitions spatially coherent?
6. Is the scene visually impressive and understandable?
7. Could a simpler structure create a more premium result?

Often the best improvement is removing unnecessary complexity and making the spatial logic stronger.

## Performance awareness

3D experiences must remain practical.

Account for:

- device performance
- loading cost
- mobile simplification
- reduced motion preferences
- fallback rendering strategies
- texture and asset weight
- responsiveness under real user conditions

The experience should feel premium without becoming fragile.

If the 3D concept is too expensive for the target device range, define a lighter mode that preserves the story and hierarchy.

## Output behavior

When asked for a concept, first describe the spatial structure and scene logic before suggesting implementation details.

When giving a recommendation, describe:

- the central object or spatial idea
- the camera behavior
- the layer structure
- the material direction
- the lighting strategy
- the transition style
- the emotional effect
- the performance implications if relevant

If the 3D idea does not improve storytelling, simplify it.
